TV D-pad Camera Controls Test

Overview

This test page demonstrates the TV D-pad camera controls functionality. The controls include directional arrow buttons for camera rotation and zoom controls.

Test Instructions

How to Test:

  1. Toggle TV Mode: Click the "Toggle TV Mode" button to simulate TV mode
  2. Direction Controls: Use arrow keys to rotate camera view
  3. Zoom Controls: Hold Enter key to zoom in/out
  4. Menu Integration: Press Escape to return to menu

Key Mappings:

  • ↑ - Rotate camera to North
  • → - Rotate camera to East
  • ↓ - Rotate camera to South
  • ← - Rotate camera to West
  • Enter - Hold to zoom in/out
  • Escape - Return to menu

Expected Behavior

  • TV Mode: When enabled, D-pad controls should appear on the left side
  • Direction Controls: Arrow keys should trigger camera rotation to cardinal directions
  • Zoom Controls: Enter key should toggle between zoom in/out states
  • Menu Integration: Escape key should hide D-pad controls and show menu
  • Responsive Design: Controls should adapt to different screen sizes
